### Breaking Changes

Some components have been updated with new APIs:

- **Button component**: New `variant` prop replaces `type`
- **Form components**: Updated validation schema
- **Theme system**: New CSS variables for customization

### Code Examples

```tsx
// Old way
<Button type="primary">Click me</Button>

// New way
<Button variant="default">Click me</Button>
```

### Migration Steps

1. Update your imports
2. Replace deprecated props
3. Test your components
4. Update your theme customizations
